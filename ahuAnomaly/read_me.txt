The ahu_anomaly function detects hard and soft faults 
	related to air handling unit operations

**Note: Please leave the arial.ttf file in the same directory as the function (.ipnyb) file. 
	This is required to generate the pictures.

This function is preset to run automatically with the sample data. If you
wish to run the function with your own data, please follow the same format and placement as
the sample files and change the input and output strings specified in the code.
